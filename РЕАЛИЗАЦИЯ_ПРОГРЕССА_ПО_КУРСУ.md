## Как работает система

### 1. Открытие урока
1. Пользователь открывает урок `/lesson/lesson-1-2`
2. `useEffect` в [`app/lesson/[lessonId]/page.tsx`](app/lesson/[lessonId]/page.tsx) вызывает `setLastLesson(lessonId, courseId)`
3. В localStorage сохраняется: `nail_last_lesson[courseId] = lessonId`

### 2. Завершение урока
1. Пользователь нажимает "Завершить и продолжить"
2. Вызывается `handleCompleteLesson()` → `addLesson(lessonId)`
3. Хук `useProgress` вызывает:
   - `addProgress(lessonId, courseId)` - добавляет урок в прогресс
   - `setLastLesson(lessonId, courseId)` - обновляет последний урок

### 3. Отображение на странице курса
1. Страница курса получает:
   - `progress: progressIds` - массив завершенных уроков
   - `getLastLessonId()` - последний просмотренный урок
2. Для каждого урока проверяется:
   - `isCompleted = progressIds.includes(lesson.id)` - показать галочку
   - `isLastLesson = lesson.id === lastLessonId` - выделить рамкой
   - `canAccess = lesson.isFree || isPurchased` - сделать кликабельным

### 4. Определение следующего урока
```typescript
// 1. Находим первый незавершенный урок
let nextLesson = allLessons.find(l => !progressIds.includes(l.id));

// 2. Если есть последний урок, берем следующий после него
if (lastLessonId) {
  const lastIndex = allLessons.findIndex(l => l.id === lastLessonId);
  if (lastIndex >= 0 && lastIndex < allLessons.length - 1) {
    const potentialNext = allLessons[lastIndex + 1];
    if (!progressIds.includes(potentialNext.id)) {
      nextLesson = potentialNext;
    }
  }
}

// 3. Если ничего не найдено, берем первый урок
if (!nextLesson) {
  nextLesson = allLessons[0];
}
```

## Преимущества реализации

✅ **Изоляция по курсам**
- Каждый курс хранит свой прогресс и последний урок отдельно
- Переход между курсами не влияет на прогресс других курсов

✅ **Обратная совместимость**
- Автоматическая миграция старых данных
- Пользователи не потеряют свой прогресс

✅ **Визуальная наглядность**
- Четкая индикация статуса каждого урока
- Легко увидеть где остановился
- Понятно какие уроки доступны

✅ **Удобство использования**
- Уроки кликабельны - можно перейти напрямую
- Кнопка "Продолжить" всегда ведет на правильный урок
- Прогресс синхронизирован везде

✅ **Локальное хранение**
- Быстрая работа без запросов к серверу
- Работает оффлайн
- Простая структура данных
