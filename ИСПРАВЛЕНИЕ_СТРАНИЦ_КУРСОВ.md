# Исправление структуры страниц курсов

## Обзор изменений

Переработана страница [`app/courses/page.tsx`](app/courses/page.tsx:1) для преобразования из личной страницы с курсами в **публичный каталог** с акцентом на продажу курсов.

## Основные изменения

### 1. ❌ Убрано требование авторизации

**Было:**
```typescript

    router.push('/');
  }
}, [isAuthenticated, authLoading, router]);

if (authLoading || !isAuthenticated) {
  return <LoadingSpinner />;
}
```

**Стало:**
```typescript
const { isAuthenticated } = useAuth();
// НЕТ редиректа и НЕТ проверки на авторизацию
// Страница доступна для всех пользователей
```

**Результат:** ✅ Страница теперь **публичная** и доступна всем посетителям

### 2. ✅ Добавлено отображение цен

**Новый функционал:**
```typescript
// Вычисляем минимальную цену из активных тарифных планов
const minPrice = course.pricingPlans && course.pricingPlans.length > 0
  ? Math.min(...course.pricingPlans.filter((p: any) => p.isActive).map((p: any) => p.price))
  : null;

const currency = course.pricingPlans[0]?.currency || '₸';
const activePlansCount = course.pricingPlans.filter((p: any) => p.isActive).length;
```

**UI для отображения цен:**
```jsx
{minPrice && !isPurchased && (
  <div className="mt-auto pt-4 border-t border-gray-200 mb-4">
    <div className="flex items-baseline justify-between mb-2">
      <span className="text-sm text-dark-600">От</span>
      <div className="flex items-baseline gap-1">
        <span className="text-2xl font-bold text-primary-600">
          {minPrice.toLocaleString()}
        </span>
        <span className="text-sm text-dark-500">{currency}</span>
      </div>
    </div>
    {activePlansCount > 1 && (
      <span className="text-xs text-dark-500">
        {activePlansCount} тарифа

)}
```

**Результат:** ✅ Пользователи видят цену **до регистрации**

### 4. ✅ Добавлена обработка покупки

**Новая логика:**
```typescript
const [selectedCourseId, setSelectedCourseId] = useState<string | null>(null);
const [showPurchaseModal, setShowPurchaseModal] = useState(false);

const handlePurchaseClick = (courseId: string) => {
  if (!isAuthenticated) {
    // Для неавторизованных: редирект на регистрацию с параметром курса
    window.location.href = `/auth/register?course=${courseId}`;
    return;
  }
  // Для авторизованных: открыть модальное окно покупки
  setSelectedCourseId(courseId);
  setShowPurchaseModal(true);
};

const handlePurchaseSuccess = () => {
  setShowPurchaseModal(false);
  window.location.reload();
};
```

**Модальное окно покупки:**
```jsx
{selectedCourseId && (
  <PurchaseModal
    isOpen={showPurchaseModal}
    onClose={() => setShowPurchaseModal(false)}
    onSuccess={handlePurchaseSuccess}
    courseId={selectedCourseId}
  />
)}
```

**Результат:** ✅ Прямой путь к покупке с любой карточки курса

## User Journey (Путь пользователя)

### Для неавторизованного посетителя:
```
1. Заходит на /courses (публичная страница)
2. Видит каталог курсов с ценами
3. Видит бейдж "Требуется регистрация"
4. Кликает "Начать обучение"
5. → Редирект на /auth/register?course={courseId}
6. После регистрации → /dashboard
7. Может купить курс или начать с бесплатных уроков
```

### Для авторизованного (не купившего):
```
1. Заходит на /courses
2. Видит каталог с ценами
3. Видит бейдж "Доступны бесплатные уроки"
4. Кликает "Купить курс"
5. → Открывается модальное окно покупки
6. Выбирает тариф и покупает
7. → Редирект на /course/{courseId} для начала обучения
```

### Для авторизованного (с прогрессом):
```
1. Заходит на /courses
2. Видит свой прогресс по начатым курсам
3. Видит бейдж "В процессе"
4. Может:
   - "Купить полный доступ" → модальное окно
   - "Подробнее о курсе" → /course/{courseId}
```

### Для купившего курс:
```
1. Заходит на /courses
2. Видит бейдж "Полный доступ"
3. Видит свой прогресс
4. Кликает "Продолжить обучение" или "Начать обучение"
5. → Переход на /course/{courseId}
```
