# 📖 Инструкция: Как редактировать модуль

## 🎯 Быстрый доступ

### Прямые ссылки на редактирование модулей:

- **Модуль 1**: http://localhost:3001/admin/modules/m1/edit
- **Модуль 2**: http://localhost:3001/admin/modules/m2/edit
- **Модуль 3**: http://localhost:3001/admin/modules/m3/edit

**Просто скопируйте и откройте в браузере!** ✅

---

## 📱 Пошаговая инструкция (На мобилке)

### Шаг 1: Войти в админку
1. Открыть: `http://localhost:3001/admin/login`
2. Ввести:
   - Email: `admin@nailacademy.com`
   - Пароль: `admin123`
3. Нажать "Войти"

### Шаг 2: Открыть список модулей
1. В **нижней навигации** найти вкладку с иконкой 📚 "Курсы"
2. Нажать на неё
3. ИЛИ открыть напрямую в меню (три полоски сверху слева)
4. Выбрать "Модули"

Или просто открыть: `http://localhost:3001/admin/modules`

### Шаг 3: Найти модуль
Вы увидите список из 3 модулей:
```

│                                    │
│ [Редактировать] [🗑️] [→]         │

┌────────────────────────────────────┐
│ #2                                 │
│ Модуль 2: Долговременные...       │
│ Ламинирование и окрашивание...    │
│ 3 уроков                          │
│                                    │
│ [Редактировать] [🗑️] [→]         │
└────────────────────────────────────┘

┌────────────────────────────────────┐
│ #3                                 │
│ Модуль 3: Продвинутые техники     │
│ Перманентный макияж и...          │
│ 3 уроков                          │
│                                    │
```

### Шаг 4: Нажать "Редактировать"
- Найти нужный модуль
- Нажать **синюю кнопку "Редактировать"** (она с иконкой ✏️)
- Откроется страница редактирования!

---

## 💻 Пошаговая инструкция (На десктопе)

### Шаг 1: Войти в админку
1. Открыть: `http://localhost:3001/admin/login`
2. Ввести данные и войти

### Шаг 2: Перейти к модулям
1. В **боковом меню слева** найти "Модули" (иконка слоёв)
2. Нажать на неё
3. Откроется список модулей

### Шаг 3: Редактировать
- Найти нужный модуль
- Нажать кнопку "Редактировать"
- На десктопе также можно **перетаскивать модули мышью** для изменения порядка!

---

## ✏️ Что можно редактировать

После открытия страницы редактирования вы увидите форму:

### 1. Название модуля
```
┌────────────────────────────────────┐
│ Модуль 1: Натуральные брови       │
└────────────────────────────────────┘
```

### 2. Описание модуля
```
┌────────────────────────────────────┐
│ Работа с натуральными бровями и   │
│ базовые техники                    │
│                                    │
│                                    │
└────────────────────────────────────┘
```

### 3. Информация
- Видно количество уроков в модуле
- Список всех уроков модуля

### 4. Кнопка сохранения
```
┌────────────────────────────────────┐
│   💾 Сохранить изменения           │
└────────────────────────────────────┘
```

## 🔍 Где находится кнопка "Редактировать"?

### На странице списка модулей (`/admin/modules`):

Каждый модуль выглядит так:

```
┌─────────────────────────────────────────────┐
│  #1                                          │
│  Модуль 1: Натуральные брови                │
│  Работа с натуральными бровями...           │
│  4 уроков                                   │
│  ─────────────────────────────────────────  │
│  [  ✏️ Редактировать  ] [ 🗑️ ] [ → ]       │  ← ТУТ!
└─────────────────────────────────────────────┘
```

**Кнопка "Редактировать" находится внизу каждой карточки модуля!**

## 🎯 Пример редактирования

1. Открыть: http://localhost:3001/admin/modules/m1/edit
2. Изменить название, например, добавить: "🌟" в начало
3. Изменить описание
4. Нажать кнопку **"Сохранить изменения"** (внизу)
5. Появится уведомление: "✅ Модуль успешно обновлён!"
6. Вернётесь к списку модулей

**Готово!** Изменения сохранены в LocalStorage и видны сразу!

---

## ❓ Если кнопка не видна

1. **Прокрутите вниз** внутри карточки модуля
2. Кнопка находится **после описания модуля**
3. Или используйте **прямые ссылки выше** ⬆️

---

## 📝 Итого

**3 способа открыть редактирование модуля:**

1. ✅ **Прямая ссылка** (самый быстрый!):
   - http://localhost:3001/admin/modules/m1/edit

2. ✅ **Через список модулей**:
   - /admin/modules → Найти модуль → Нажать "Редактировать"

3. ✅ **Через меню**:
   - Меню → Модули → Найти модуль → Редактировать

**Кнопка "Редактировать" есть! Она синяя, внизу каждой карточки модуля!** 🎉
