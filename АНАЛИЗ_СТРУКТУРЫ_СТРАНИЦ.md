## Проблемы и несоответствия

### Основная проблема: `/courses` - неправильное позиционирование

**Что сейчас:**
```
/ (публичная landing) → /auth/login → /courses (требует авторизацию) → /course/[id] → покупка
```

**Что должно быть:**
```
/ (публичная landing) → /courses (ПУБЛИЧНЫЙ каталог) → /course/[id] → покупка → /auth/register
                                                                              ↓
                                                                         /dashboard
```

### Детальные проблемы `/courses`:

1. **Требование авторизации** ❌
   - Пользователи не могут увидеть курсы БЕЗ регистрации
   - Это уменьшает конверсию
   - Классический антипаттерн в e-commerce

2. **Отсутствие ценовой информации** ❌
   - Не показывает цены на курсы
   - Не понятна ценность предложения
   - Нет мотивации к покупке

3. **Нет маркетинговых элементов** ❌
   - Нет призывов к действию (CTA)
   - Нет социальных доказательств
   - Нет выгод от покупки

4. **Путаница с назначением** ❌
   - Сейчас это просто список курсов
   - Не понятно, зачем нужна эта страница отдельно от `/dashboard`
   - Дублирует функционал дашборда

**Цель:** Привлечение и конверсия в покупку
