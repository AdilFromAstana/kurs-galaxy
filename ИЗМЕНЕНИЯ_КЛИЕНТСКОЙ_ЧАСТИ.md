### 3. ✅ Хук покупок ([`hooks/usePurchase.ts`](hooks/usePurchase.ts))

```typescript
// ❌ Было
```

## 🔧 Технические детали

### Миграция данных

Автоматическая миграция старых данных в [`lib/storage.ts`](lib/storage.ts:56):
```typescript
export const getAllProgress = (): ProgressByCourse => {
  const parsed = JSON.parse(data);
  
  // Миграция старого формата (массив) в новый (объект)
  if (Array.isArray(parsed)) {
    const migrated = { 'brow-master-pro': parsed };
    localStorage.setItem('nail_progress', JSON.stringify(migrated));
    return migrated;
  }
  
  return parsed;
}
```

### Обратная совместимость

Хуки сохраняют обратную совместимость с дефолтными значениями:
```typescript
// useProgress - courseId по умолчанию 'brow-master-pro'
export const useProgress = (courseId: string = 'brow-master-pro') => { ... }

// Но usePurchase - строго требует courseId (правильно!)
export const usePurchase = (courseId: string) => { ... }
```

## 🎓 Как использовать

### Для добавления нового курса:

1. Добавить курс в [`lib/courseData.ts`](lib/courseData.ts:4):
```typescript
export const coursesData: Course[] = [
  { id: 'brow-master-pro', ... },
  { id: 'nail-art-pro', ... }, // НОВЫЙ КУРС
];
```

2. Курс автоматически появится:
   - На главной странице (в счетчиках)
   - На [`/courses`](app/courses/page.tsx) (в списке)
   - Будет доступен по [`/course/nail-art-pro`](app/course/[courseId]/page.tsx)

3. Прогресс будет автоматически раздельным

### Для студента:

1. [`/courses`](app/courses/page.tsx) - увидеть все курсы
2. [`/course/[courseId]`](app/course/[courseId]/page.tsx) - посмотреть программу курса
3. Купить доступ через [`PurchaseModal`](components/modals/PurchaseModal.tsx)
4. [`/dashboard`](app/dashboard/page.tsx) - начать обучение
