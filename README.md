# 💅 KursGalaxy.kz - LMS платформа для обучения

Полнофункциональная система управления обучением (LMS) для онлайн-курса маникюра с использованием Next.js 14, TypeScript и Tailwind CSS.

## ✨ Особенности

- ✅ **Mobile First дизайн** - оптимизировано для мобильных устройств
- ✅ **Offline-first** - все данные хранятся в LocalStorage
- ✅ **Полный цикл обучения** - от регистрации до получения сертификата
- ✅ **Система прогресса** - визуальное отслеживание завершенных уроков
- ✅ **Видео-плеер** - с сохранением позиции и контролем скорости
- ✅ **Модульная структура** - 3 модуля, 10 уроков
- ✅ **Premium дизайн** - розовые, пастельные тона
- ✅ **Плавные анимации** - Tailwind CSS анимации

## 🚀 Быстрый старт

### Установка зависимостей

```bash
cd nail-academy-pro
npm install
```

### Запуск в режиме разработки

```bash

```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

### Сборка для продакшна

```bash
npm run build

```

## 🎯 Функциональность

### 1. Авторизация
- Регистрация с именем, email и паролем
- Вход в систему
- Автоматическое перенаправление на Dashboard

### 2. Dashboard
- **ProgressWidget** - круговая диаграмма прогресса
- **ContinueButton** - быстрый переход к следующему уроку
- **ModuleList** - список всех модулей и уроков с статусами

### 3. Страница урока
- **VideoPlayer** с:
  - Сохранением позиции воспроизведения
  - Контролем скорости (0.5x - 2x)
  - Адаптивным дизайном
- **Конспект урока** - структурированный текст
- **Материалы** - ссылки на PDF и внешние ресурсы
- **Навигация** - переход к следующему уроку

### 4. Система покупки
- Модальное окно с описанием преимуществ
- Имитация успешной оплаты
- Разблокировка всех уроков после покупки

### 5. Профиль
- Информация о пользователе
- Статистика прогресса
- Сертификат (доступен при 100% завершении)

## 🎨 Дизайн

### Цветовая палитра
- **Primary (Розовый)**: `#ec4899` - основной акцентный цвет
- **Dark**: `#18181b` - текст и темные элементы
- **Фон**: Градиент от `primary-50` до `white`

### Mobile First подход
Все компоненты разработаны с учетом мобильных устройств:
- Базовые стили для мобильных (до 640px)
- `md:` для планшетов (от 768px)
- `lg:` для десктопов (от 1024px)

### Анимации
- `animate-fade-in` - плавное появление
- `animate-slide-up` - появление снизу вверх
- `animate-scale-in` - масштабирование

## 🎓 Содержание курса

### Модуль 1: Основы маникюра (4 урока)

9. Наращивание гелем
